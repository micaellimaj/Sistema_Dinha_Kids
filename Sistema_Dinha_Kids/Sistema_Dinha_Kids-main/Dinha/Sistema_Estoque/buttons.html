Brand Buttons

Os botões do Google e do Facebook estão disponíveis com os respectivos de cada empresa cor da marca.Eles são usados nas páginas de login e registro do usuário.

YVocê pode criar mais botões personalizados adicionando uma nova variável de cor no _variables.scss Arquivo e, em seguida, usando a variante do botão Bootstrap Misture para criar um novo estilo, como demonstrado no _buttons.scss file.

.btn-google .btn-facebook
